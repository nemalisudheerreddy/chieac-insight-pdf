# Project Submission Summary

## Project name
ChiEAC InsightPDF — Community Document Intelligence Assistant

## Created by
Sudheer Reddy Nemali

## Purpose
The application helps users understand long or complex PDF documents by allowing them to ask natural-language questions and receive concise, document-grounded answers.

## Community value
ChiEAC and its community can use the tool to make educational resources, policy documents, program guides, and administrative materials easier to navigate. The source-excerpt view supports transparency and helps users verify each answer.

## Technical implementation
- React and Vite frontend
- FastAPI and Python backend
- PyPDF text extraction
- OpenAI embeddings
- NumPy similarity search
- OpenAI answer generation
- Automatic isolated browser sessions
- Page-level citations and supporting excerpts
